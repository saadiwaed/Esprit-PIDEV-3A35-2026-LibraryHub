# LibraryHub – Smart Library Management System

## Overview

This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering – Tunisia** (Academic Year 2025–2026).

LibraryHub is a web platform that helps manage library resources such as books, users, and borrowing activities. The system integrates a Java-based AI module to recommend books to users.

## Features

- Book management
- User management
- Borrow and return books
- Library catalog browsing
- AI-based book recommendation system
- journal and defi
- post and forum 
- club and events
